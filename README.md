# Python
Number Based Python Programs
1)	Palindrome Number – A number that reads the same forward and backward.
Example: 121, 1331
2)	Armstrong Number – A number equal to the sum of its digits raised to the power of the number of digits.
Example: 153 = 1³ + 5³ + 3³
3)	Fibonacci Series – A sequence where each number is the sum of the two previous numbers.
Example: 0, 1, 1, 2, 3, 5, 8
4)	Factorial of a Number – The product of all positive integers up to that number.
Example: 5! = 5 × 4 × 3 × 2 × 1 = 120
5)	Prime Number Check – A number that has only two divisors: 1 and itself.
Example: 7
6)	Prime Numbers in a Range – All numbers in a range that are prime numbers.
Example (1–10): 2, 3, 5, 7
7)	Perfect Number – A number equal to the sum of its proper divisors excluding itself.
Example: 6 = 1 + 2 + 3
8)	Strong Number – A number equal to the sum of the factorials of its digits.
Example: 145 = 1! + 4! + 5!
9)	Reverse a Number – Writing the digits of a number in reverse order.
Example: 123 → 321
10)	Sum of Digits of a Number – The total obtained by adding all the digits of a number.
Example: 123 → 1 + 2 + 3 = 6
